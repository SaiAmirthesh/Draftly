# Draftly - AI Email Assistant

Draftly is a simple, AI-powered email drafting tool that helps you create professional emails in seconds.


## Features

- AI-powered email generation using Google's Gemini
- Multiple email types: Business, Cold, Cover Letters, Follow-ups, etc.
- Tone customization: Casual, Professional, Formal, Friendly, Persuasive
- Easy copy-paste: Generated emails ready to use
- Improve & regenerate: Get better versions instantly

## How to Use

1. Select email type (Business, Cold, Cover Letter, etc.)
2. Choose tone (Casual, Professional, etc.)
3. Enter details:
    - Recipient name/role
    - What you want to say
    - Your name/role
4. Click "Generate Email"
5. Copy & paste the generated email
